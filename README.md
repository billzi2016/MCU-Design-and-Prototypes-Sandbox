# MCU Design and Prototypes Sandbox

这是本人设计和整理的 MCU / Arduino / 嵌入式系统原型方案合集，覆盖电子信息、自动化、嵌入式、控制、机器人和智能感知等方向。

项目目标不是做简单点灯实验，而是围绕传感器采集、显示交互、通信定位、电机控制、闭环控制、机器人执行机构和视觉感知，整理一批可以逐步实现、测试和扩展的原型系统。

## 项目定位

- 仅使用 Arduino Uno、Arduino Mega 2560、ESP32、ESP8266 作为主要实现平台。
- 每个项目选择能满足需求的最小平台，不为了复杂而使用更高规格硬件。
- 每个项目应具备明确输入、处理逻辑、输出执行或显示结果。
- 简单项目也要做成完整系统，而不是单个模块演示。
- 复杂项目应体现控制逻辑、数据处理、通信协议、机械结构或视觉算法。
- 项目 README、代码注释、接线说明和测试记录应使用中文。

## 基础控制与显示类

### 1. 篮球计分器

- 主要器件：Arduino、按键、数码管或 LCD、蜂鸣器、LED。
- 核心功能：两队比分显示、加分、减分、节次显示、倒计时、暂停计时、蜂鸣提示。
- 扩展方向：无线遥控计分、24 秒进攻计时、犯规次数统计、OLED 菜单。

### 2. 多功能数码管电子钟

- 主要器件：Arduino、DS3231、四位或八位数码管、按键、蜂鸣器。
- 核心功能：时间显示、日期显示、闹钟、倒计时、按键校时。
- 扩展方向：温度显示、断电保持、亮度自动调节。

### 3. OLED 菜单交互系统

- 主要器件：Arduino 或 ESP32、OLED、旋钮编码器、按键。
- 核心功能：多级菜单、参数设置、状态显示、页面切换。
- 扩展方向：作为其他项目的人机交互模块。

### 4. 交通灯控制系统

- 主要器件：Arduino、LED、数码管、按键、蜂鸣器。
- 核心功能：红绿灯周期控制、人行道按钮、倒计时显示。
- 扩展方向：车流量检测、自适应信号灯控制、应急车辆优先通行。

### 5. 电子密码锁

- 主要器件：Arduino、矩阵键盘、舵机或电磁锁、LCD、蜂鸣器。
- 核心功能：密码输入、密码修改、错误报警、开锁控制。
- 扩展方向：RFID 解锁、指纹模块、蓝牙临时密码。

### 6. 智能台灯控制系统

- 主要器件：Arduino、光敏电阻、LED 灯带、PWM 驱动模块、按键。
- 核心功能：亮度检测、自动调光、手动调光、模式切换。
- 扩展方向：人体感应、定时关闭、OLED 状态显示。

## 后续组织方式

后续可以按照项目逐个创建子目录。每个子项目目录只保留必要文件：

```text
project-name/
├── README.md
├── project-name.ipynb
└── src/
```

每个子项目至少说明：

- 项目目标
- 器件清单
- 接线说明
- 核心代码
- 测试方法
- 可扩展方向
